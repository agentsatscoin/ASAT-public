# Screenshots

This folder is for carefully selected public-safe screenshots only.

Good examples:
- homepage sections
- public-facing protocol surfaces
- roadmap or architecture visuals
- sanitized UI proof that the project exists and is active

Do not upload screenshots that expose:
- admin secrets
- guardian identifiers tied to sensitive context
- internal API responses
- private routes or tokens
- hidden infrastructure details
- private operational logic

Recommended public screenshot set:
1. homepage hero
2. public registry/task direction surface
3. roadmap section
4. sanitized admin control room shell only, if it reveals nothing sensitive
